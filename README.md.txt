# MoodBoard Roulette 🎨🧠

**MoodBoard Roulette** is an emotion-aware fullstack web app that helps users explore their feelings through text or speech and receive personalized content or mental health support.

## 🌟 Features
- 🎤 Voice or text-based emotion input
- 🧠 Sentiment analysis (using Sentiment.js)
- 🎨 Mood-based content inspiration from the Unsplash API
- 🆘 Redirects users to free therapy/help resources if feeling low
- 🧑‍🤝‍🧑 Suggests social connection apps for loneliness
- 📖 Mood journal history (coming soon)
- 📚 Emotional education based on user input

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- APIs: Sentiment.js, Unsplash API, Web Speech API
